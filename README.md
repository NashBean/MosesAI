# MosesAI - iBeanSoftware Knowledge Base on Biblical Moses

This repo holds comprehensive knowledge on Moses: readings/writings (Exodus-Deuteronomy), path (Egypt to Sinai to Jordan), archaeology (Merneptah Stele, Serabit el-Khadim inscriptions), and more. Integrated with in-house BDH GAI for queries + watsonx for advanced AI insights.

## Mission
To share true knowledge of Moses as God's chosen leader, humbly guiding Israel through miracles and law—showing AI can respectfully explore Scripture for deeper faith, without profit.

## Data Sources
- Path: Born in Egypt (Exodus 2), Midian exile, burning bush call (Exodus 3-4), plagues/Exodus (1446 BC), Sinai covenant, wilderness (40 years), death at Jordan.
- Archaeology: Merneptah Stele (1208 BC "Israel is laid waste"—earliest mention), Serabit el-Khadim inscriptions (1800 BC possible "Moses" reference), Hatshepsut (Pharaoh's daughter candidate), Pi-Ramesses (biblical Ramses city).
- Writings/Readings: Exodus-Deuteronomy (authored by Moses per tradition), Psalms 90 (Moses' prayer). Historical: 15th century BC aligns with Amarna letters/Canaan conquest.

See data/moses_knowledge.json for full DB.

## Usage
- Query: python3 scripts/moses_query.py "Moses' miracles at Sinai"
- Watsonx: Enhanced insights via IBM API.

Created by iBeanSoftware—showing good AI can honor God's Word.


## Run
- Console: `python3 main.py`
- Server: `python3 MosesAI_Server.py`
- C++ Console: compile `MosesAI_Console.cpp` and run `./MosesAI_Console`
